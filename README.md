Nuestro proyecto final, llamado NYSO, trata de una aplicación sobre vinos, la cual incluye varias funcionalidades: un buscador de vinos con filtros avanzados y buscador, un blog con noticias con la posibilidad de enviar recomendaciones si estas registrado, un apartado de red social donde podrás encontrar y conectar con diferentes personas o amigos, la posibilidad de registrarse, creando asi un perfil con preferencias sobre vinos, lista de historial y favoritos. Ademas, generamos un panel de administrador (le dejamos los detalles en el txt para poder ingresar) para poder gestionar el blog, tanto las recomendaciones como las entradas ya existentes. 

Las tecnologías que utilizamos son Vue, JavaScript, Supabase, Vite y Tailwind.
Para ejecutar e instalarlas hay que correr los siguientes comandos:
npm install
npm run dev
